
# Limitless Outdoor Living Website

A responsive business website built for **Limitless Outdoor Living**, a Metro Atlanta outdoor living company focused on luxury backyard transformations, cabana restoration, sunroom/screen porch renovation, deck and patio upgrades, and custom outdoor carpentry.

## Live Website

https://limitlessoutdoorliving.netlify.app

## Project Overview

This website was designed to give the business a professional online presence and help convert visitors into leads through clear service sections, strong visuals, mobile-friendly design, and direct contact options.

The site includes a modern landing page layout, background video sections, project showcases, testimonials, call/text/email buttons, and a working contact form powered by Formspree.

## Features

- Responsive mobile-friendly layout
- Sticky navigation bar
- Hero section with background video
- Service cards with icons
- Featured project showcase
- Before-and-after transformation section
- Why Choose Limitless section
- Client testimonial section
- Click-to-call, click-to-text, and click-to-email contact cards
- Formspree contact form integration
- Custom favicon
- SEO meta tags
- Sitemap for Google and Bing indexing
- Google Search Console verification
- Bing Webmaster setup

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Formspree
- Netlify
- Lucide Icons
- Google Fonts

## SEO Work Included

- Optimized page title and meta description
- Local keywords for Metro Atlanta outdoor living services
- Open Graph tags for link previews
- Sitemap.xml setup
- Google Search Console submission
- Bing Webmaster Tools submission
- Location-focused content for local search visibility

## Contact Functionality

The contact section allows users to:

- Call the business directly
- Send a text message
- Open an email draft
- Submit a project request through the contact form

## Project Purpose

This project was built as a real-world client website to practice front-end development, responsive design, SEO basics, lead-generation design, and business-focused web deployment.

## Deployment

The website is deployed through Netlify using manual folder deployment.
